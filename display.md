# Display

> Display는 element가 block 또는 inline인지를 결정하거나 child element의 레이아웃을 결정한다.(grid, flex, flow layout)



## Display Syntax

```css
.container {
  display:  [ <display-outside> || <display-inside> ] | <display-listitem> | <display-internal> | <display-box> | <display-legacy> ;
}
```
